## ‚ö° Quick Fixes

If something looks off:
1. **Hard refresh**: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
2. **Clear cache**: DevTools ‚Üí Application ‚Üí Clear Storage
3. **Restart dev server**: Ctrl+C then `npm run dev`
